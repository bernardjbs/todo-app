# Documentation Rules (Global)

## Language

- Use UK English throughout (e.g. "colour", "organise", "behaviour", "analysed")
- This applies to all documentation, comments, commit messages, PR descriptions, and skill/agent definitions

## Journey Summaries

- Location: `docs/summaries/NNN_summary_<topic>.md`
- Writing style: first-person, humble tone, natural language
- Proper punctuation — no dashes as sentence punctuation
- Not AI-sounding
